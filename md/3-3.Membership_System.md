## 👥 Membership System (회원 체계)

### 1. 회원 구조도

```
가입 회원
├── 판매 회원(복수선택 가능)
│   ├── 시행사
│   ├── 대행사
│   ├── 공인중개사
│   └── 분양인
└── 구매 회원(복수선택 가능)
    ├── 건물주
    ├── 인테리어
    ├── 법무사
    ├── 세무사
    ├── 금융인
    └── 변호사
```

### 2. 회원 종류

| 회원 종류   | 설명                          | 권한                       | 비고                                          |
| ----------- | ----------------------------- | -------------------------- | --------------------------------------------- |
| 관리자      | 시스템 관리                   | 모든 권한                  |                                               |
| 판매 회원   | 분양/전매/매매/임대 하는 회원 | 판매 회원,구매 회원 페이지 | 시행사, 대행사, 공인중개사, 분양인(중복 가능) |
| 구매 회원   | 기본 사용자                   | 구매 회원 페이지           | 건물주,인테리어,법무사,세무사,금융인,세무사   |
| 광고매 회원 | 광고를 원하는 회원            | 광고 페이지                |                                               |

### 4. 가입 절차

- **관리자**: admin 임명
- **판매 회원**: 명함 입력으로 관리자 승인
- **구매 회원**: 일반 가입
